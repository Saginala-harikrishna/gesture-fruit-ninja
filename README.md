# Gesture Gaming Hub 🎮✋

An interactive **AI-powered gesture gaming system** built using **Computer Vision** and **Hand Tracking** technologies, where users can play games **without touching the keyboard or mouse**.

This project was successfully demonstrated during our college fest, where students interacted with the games using only **air hand gestures**.

---

# 🚀 Features

## 🍉 Gesture Fruit Ninja

* Slice fruits using hand gestures
* Real-time hand tracking
* Dynamic fruit slicing effects
* Juice splash animations
* Bomb collision effects
* Score tracking system
* Timer-based gameplay
* Gameplay recording support

---

## 🏎️ Gesture Car Racing

* Control the car using hand movements
* Real-time steering detection
* Border collision system
* Obstacle avoidance mechanics
* Score calculation system
* Smooth gesture-based navigation

## 🏎️ Car Racing

# 🏅 Scoring System

## Fruit Ninja

| Action      | Score |
| ----------- | ----- |
| Slice Fruit | +1    |

## Car Racing
